# WEATHER MONSTER

API for different institutions to monitor weather temperature and get forecasts. 

## Setup

-   Node version: 14.15
-   run `npm ci`

## Running the tests
-   run `npm run test`

## Running the app
-   run `npm start`

## EndPoints
The API has four endpoints: 

● _`/cities`_ 

● _`/temperatures`_

● _`/forecasts`_

● _`/webhooks`_
